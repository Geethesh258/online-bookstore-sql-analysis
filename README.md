# Online Bookstore Sales Analysis (PostgreSQL)

## Project Overview
This project analyzes bookstore sales data using PostgreSQL to gain insights into revenue, customer behavior, and inventory status.

## Tools Used
- PostgreSQL
- SQL
- GitHub

## Database Tables
- Books
- Customers
- Orders

## Key Insights
- Total revenue and inventory analysis
- Genre-wise and author-wise sales
- Top customers and most sold books
- Remaining stock after fulfilling orders

## Skills Demonstrated
SQL Joins, Aggregate Functions, GROUP BY, HAVING, Data Analysis
